What does self refer to inside a class's method?
**Short answer:**
`self` refers to the **current object (instance)** of the class.

---

### Slightly longer (interview-ready) explanation:

> Inside a class method, `self` represents the **specific object that is calling the method**. It allows the method to access and modify that objectâ€™s attributes and other methods.

---

### Example

```python
class Person:
    def __init__(self, name):
        self.name = name

    def greet(self):
        print("Hello, my name is", self.name)
```

```python
p1 = Person("Alice")
p2 = Person("Bob")

p1.greet()
p2.greet()
```

* When `p1.greet()` is called â†’ `self` refers to `p1`
* When `p2.greet()` is called â†’ `self` refers to `p2`

So:

```python
self.name
```

means:

```python
p1.name   # or p2.name, depending on caller
```

---

### Important clarifications (very common confusion)

* `self` is **not a keyword** (you could name it something else, but donâ€™t)
* Python **passes it automatically**
* It links **methods to object data**
